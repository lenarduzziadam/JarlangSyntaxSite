üõ†Ô∏è Installation Guide

Install Java Development Kit (JDK)

Jarlang requires Java to run. Download and install JDK 11 or higher:

  • Oracle JDK: oracle.com/java
  • OpenJDK: openjdk.org
  • Amazon Corretto: aws.amazon.com/corretto

Verify installation by running:

java -version
javac -version

Download Jarlang Runner

Download the Jarlang JAR file from the Google Drive link above. The main file you need is:

jarlang-runner.jar

Place this file in a dedicated folder like C:\jarlang\ or ~/jarlang/

Run Your First Jarlang Program

Create a simple Jarlang program (hello.jar):

vessel main() {
echo("Hello, warrior jar world!");
}

Run it with the Jarlang interpreter:

java -jar jarlang-runner.jar hello.jar

Set Up Environment (Optional)

For easier access, add Jarlang to your system PATH or create an alias:

Windows (PowerShell):

function jarlang { java -jar C:\path\to\jarlang-runner.jar $args }

macOS/Linux (Bash/Zsh):

alias jarlang='java -jar ~/path/to/jarlang-runner.jar'

Then you can run programs with just:

jarlang hello.jar

üîß Troubleshooting

Common Issues

  • "java: command not found"
    Java is not installed or not in PATH. Install JDK and restart terminal.
  • "Could not find or load main class"
    JAR file is corrupted or path is incorrect. Re-download the JAR file.
  • "UnsupportedClassVersionError"
    Java version is too old. Update to JDK 11 or higher.
  • Jarlang syntax errors
    Check the syntax documentation for proper Jarlang code structure.
