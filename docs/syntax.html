What is Jarlang?

Jarlang is a small, warrior-themed expression language and learning interpreter implemented in Java. It uses themed keywords (forge, wield, chant, summon, judge, lest, mend) and provides familiar arithmetic, control flow, functions, and a small standard library.

Lexical Elements

  • Comments: # single-line comment
  • Identifiers: Letters, digits, and _, must start with a letter
  • Strings: "Hello"
  • Numbers: 42, 3.14
  • Tokens:
    • Arithmetic: + - * / ^
    • Comparison: == != < > <= >=
    • Assignment: =
    • Grouping: ( ) { }
    • Delimiters: , : ;

Themed Keywords

  • wield ‚Äî declare a variable (wield x 10)
  • forge ‚Äî begin a function definition (forge add(a, b) ...)
  • mend ‚Äî return from a function or end a single-line judge branch
  • chant ‚Äî print/output expression
  • summon ‚Äî import another .vase file
  • judge ‚Äî if/conditional branch
  • orjudge ‚Äî else-if/chained conditional branch
  • lest ‚Äî while loop
  • gather/disperse ‚Äî parentheses (internal naming)

Statements & Control Flow

  • Variable declaration: wield x 10
  • Assignment: x = x + 1
  • Print: chant "Hello"
  • Blocks: { ... } for grouped statements
  • If / elif / else:
    • judge x > 0 mend ...
    • orjudge x == 0 mend ...
    • orjudge mend ...
  • While loop: lest i < 5 { ... }
judge x > 0 mend chant "positive" orjudge x == 0 mend chant "zero" orjudge mend chant "negative"

Functions

  • Define:
    • forge add(a, b) { mend a + b }
  • Return:
    • mend result
  • Call:
    • chant add(2, 3)

Imports

summon "stdlib.vase" executes another file and injects top-level definitions into current context.
summon "stdlib.vase" chant commune(3, 4) # 7.0

Examples

Conditional and loops:
wield n 5 judge n <= 1 mend chant "trivial" orjudge mend wield i 0 lest i < n { chant i i = i + 1 }
